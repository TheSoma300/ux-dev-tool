The plan

Business Requirements

The tool needs to be a web-app that the compay can host within its organization's VPN.

It needs to have an export functionality for all the files and modules required to render the page.

The code needs to be adhering to code quality standards of the company.

Design Decisions

The UX team needs a drag and drop feature for the components within a grid based placement (css-grids).

As it is a design tool, the layout should follow common norms, in terms of tool placements (togglable side-menu).

Dev Decisions

The web-app will be delivered using Angular 10 on the front-end, and ASP.NET Core 3.1 in the back-end.

For the drag and drop feature, the Dragula npm library will be used.
